# Northstar Finance Dashboard

Single-page personal finance dashboard built with React, TypeScript, Tailwind CSS, and Recharts.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Features

- Header with greeting, current date, and persistent dark/light mode toggle
- Summary cards for income, expenses, savings, and net worth
- Trend line chart for income versus expenses
- Expense mix donut chart
- Sortable and filterable recent transactions table
- Budget pacing cards with threshold-based progress states
- Responsive layout tuned for mobile through desktop
